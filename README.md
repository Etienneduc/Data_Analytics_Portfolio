**Path:** `Project_onclass/churn_in_Telcom_Using AI python/`

**Key Processes**
- Data Cleaning, Feature Engineering & EDA

📈 **Goal:** Provide insights for customer retention strategies.  
🧰 **Tools:** Python, Pandas, Scikit-learn, Seaborn, Matplotlib
